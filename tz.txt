1. Призначення системи
Менеджер паролів - консольна утиліта для зберігання облікових даних користувача.

2. Функціональні вимоги
- Додавання запису (сервіс, логін, пароль)
- Перегляд усіх записів
- Пошук запису за сервісом
- Редагування запису
- Видалення запису
- Сортування записів
- Автоматичне збереження дат створення та оновлення

3. Формат даних
Файл passwords.json.
Поля: service, login, password, created_at, updated_at.

4. Нефункціональні вимоги
- Мова програмування Python
- Формат зберігання JSON
- Консольний інтерфейс
- Обробка помилок введення

5. Обмеження
- Локальне зберігання даних
- Без мережевої синхронізації
- Без багатокористувацького режиму
